# Network Security Sampling Checklist

Use this checklist during audit fieldwork.

## Firewall Rules
☐ Rule ID  
☐ Source / Destination  
☐ Port / Protocol  
☐ Business justification documented  
☐ Approval evidence  
☐ Last review date  
☐ Expiry date (if temporary)

## Network Segmentation
☐ Network zone defined  
☐ Access paths documented  
☐ Firewall enforcement verified  
☐ Exception documented (if any)

## VPN Users
☐ User ID  
☐ Role / business function  
☐ MFA enabled  
☐ Access scope appropriate  
☐ Logging enabled

## IDS / IPS Alerts
☐ Alert ID  
☐ Timestamp  
☐ Detection signature  
☐ Triage notes present  
☐ Escalation decision justified  
☐ Closure evidence

## Logging & Monitoring
☐ Logs ingested into SIEM  
☐ Retention meets policy  
☐ Alerts mapped to MITRE ATT&CK
