## Scope Definition (Example)

In scope:
- Identity & access management (IAM)
- Logging & monitoring (SIEM / use cases)
- Vulnerability management lifecycle
- Cloud security governance (policies, configurations oversight)
- Incident response governance

Out of scope (example):
- red team operations
- application source code review (unless requested)

Key stakeholders:
- CISO / ISO
- IT Ops / Cloud Ops
- SOC lead
