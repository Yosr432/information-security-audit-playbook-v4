# CIS Critical Security Controls Mapping

This document aligns ISO/IEC 27001:2022 Annex A controls
with relevant CIS Critical Security Controls.

| ISO 27001 Annex A | Control | CIS Control |
|------------------|---------|-------------|
| A.5.15 | IAM and access control | CIS 6 – Access Control Management |
| A.8.5 | MFA enforcement | CIS 6.3 |
| A.8.15 | Logging and monitoring | CIS 8 – Audit Log Management |
| A.8.13 | Backup and restore | CIS 11 – Data Recovery |
| A.8.9 | Secure configuration | CIS 4 – Secure Configuration |

This mapping enables technical benchmarking against industry best practices.
