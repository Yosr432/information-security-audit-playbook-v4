# NIST Cybersecurity Framework Mapping

This document maps ISO/IEC 27001:2022 Annex A controls used in this project
to the NIST Cybersecurity Framework (CSF) functions and categories.

| ISO 27001 Annex A | Control Description | NIST CSF Function | NIST CSF Category |
|------------------|--------------------|------------------|------------------|
| A.5.15 | Identity and access management | Protect | PR.AC |
| A.8.15 | Logging and monitoring | Detect | DE.CM |
| A.8.13 | Backup and recovery | Recover | RC.RP |
| A.8.9 | Secure configuration | Protect | PR.IP |

This mapping supports cross-framework assurance and governance reporting.
